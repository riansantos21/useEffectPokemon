# useEffectPokemon

##  Sobre

Este projeto foi desenvolvido em **React Native** utilizando **Expo**. A aplicação consome a **PokéAPI** para listar Pokémon e permitir a busca por seus nomes.

##  Funcionalidades

- Listar Pokémon
- Buscar Pokémon por nome
- Exibir informações dos Pokémon
- Interface simples e responsiva

##  Tecnologias

- React Native
- Expo
- TypeScript
- React Hooks (`useEffect` e `useState`)
- PokéAPI

##  Como executar

```bash
git clone https://github.com/riansantos21/useEffectPokemon.git

cd useEffectPokemon

npm install

npx expo start
```
